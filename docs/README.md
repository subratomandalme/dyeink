# DyeInk
A simple to use, fast, and minimalistic blogging site.
## Live
[![Live](https://vercel.com/button)](https://dyeink.subratomandal.com)
## Features

*   **`Writing`**: A minimalistic text editor that gets out of your way
*   **`Design`**: A carefully crafted dark mode and responsive layout
*   **`Identity`**: Connect your own custom domain easily
*   **`Security`**: Secure authentication and data protection by default
*   **`Performance`**: Built on React and Vite for instant page loads

## Techology Stack

I used industry-standard technologies to keep the codebase maintainable and scalable.
* **Frontend**
  * `Framework`: React built with Vite
  * `Language`: TypeScript
  * `Styling`: Tailwind CSS
* **State Management:** Zustand for global client state
* **Routing:** React Router
* **Backend & Database**
  * `Backend`: Supabase
  * `Database`: PostgreSQL
  * `Auth`: Supabase Auth (Persistence managed via local storage)
* **Infrastructure & Rendering**
  * `Host`: Vercel
  * `Containerization`: Docker
